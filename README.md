## How to enable GitHub Pages
1. Commit `index.html` to the repository root.
2. Open the repository on GitHub → Settings → Pages (or Settings → Code and automation → Pages).
3. Under "Build and deployment" choose Branch: `main`, Folder: `/ (root)`, Save.
4. Wait a minute. The site will be available at `https://mharisbilal-hub.github.io`.
